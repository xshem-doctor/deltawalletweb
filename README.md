# Delta Wallet Web

Simple React + Vite website for Delta Wallet.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

The development server runs with Vite. The public routes are:

- `/`
- `/en`
- `/ar`
- `/en/product`
- `/ar/product`
- `/en/services`
- `/ar/services`
- `/en/security`
- `/ar/security`
- `/en/about`
- `/ar/about`
- `/en/contact`
- `/ar/contact`
- `/en/faq`
- `/ar/faq`
- `/en/privacy`
- `/ar/privacy`
- `/en/terms`
- `/ar/terms`
- `/en/risk`
- `/ar/risk`
